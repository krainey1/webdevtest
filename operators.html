Operators

Now that you can print (at least a little) let's talk about operators. Operators are the symbols that tell the compiler to do a specific operation. Like the arithmetic operators that we'll start with first.

For instance the + is the operator for addition (Notice we don't use quotation marks here, we are not declaring the string object but rather numbers)

console.log(2 + 2); (would return 4)

And - is the symbol for subtraction

console.log(4 - 2);(would return 2)

The asterisk * is used for multiplication

console.log(2 * 2); (would return 4)

And the foward slash / is used for division

console.log(4 / 2); (would return 2)

Another operator worth noting is modulus % which returns the remainder of a value

console.log(5 % 2); (would return 1)

There are also increment operators ++ and -- that can increment a value by adding 1 or subtracting 1. This will become more important later when working with variable values.

Got all that? Awesome. There are a few more operators you might want to become familiar with later, such as assignment operators for variables. (we'll get to it soon!)

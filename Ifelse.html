Working With Conditional Statements & Comparison Operators

In JavaScript (&other coding languages!) you may want to use a Conditional Statement. A conditional statment is a piece of code that executes if x condition is met. We do this in JavaScipt using if, else, and else if statements.

First however, it is important to know how to use comparison operators. So what they are and their uses are listed below.

== (equal to)
=== (equal value & type)
!= (not equal to)
!== (not equal value & type)
> (greater than)
< (less than)
>= (greater than or equal to)
<= (less than or equal to)

Alright so now lets move onto the if statement. It's structured like this:

if(condition)
{
  code executes if condition is met
}
So it will look like this in practice:
if(var == true)
{
  console.log("true");
}

Now that we've got that down we can also add else and else if statements. They basically add another line of reasoning. So for instance if x first condition is not met then run this piece of code instead.

Here's how to use the else in a if/else statement:
if(color != "yellow")
{
  console.log("It's not yellow");
}
else
{
  console.log("It's Yellow");
}
And you can also contruct if/ else if / else statments:
if(color != "yellow")
{
  console.log("It's not yellow");
}
else if(color == "green")
{
  console.log("It's Green");
}
else
{
  console.log("It's Yellow");
}

And that's all for conditional statments. How about we look at loops?
